# ☕ Programação Orientada a Objetos (POO)

Este repositório é dedicado ao registro de aulas, exercícios e projetos desenvolvidos durante o **3º semestre** da graduação em **Ciência da Computação**. O objetivo é documentar o aprendizado dos princípios da POO e a aplicação prática utilizando a linguagem Java.

---

## 🏛️ Instituição
- **Universidade:** Universidade Franciscana (UFN)
- **Disciplina:** Programação Orientada a Objetos

## 🛠️ Tecnologias e Ferramentas
- **Linguagem:** Java (JDK 17 ou superior)
- **IDE:** VS Code / IntelliJ IDEA / Apache
- **Controle de Versão:** Git & GitHub
